About this page

This page has been created in the result of project on Silesian University of Technics. This site is hosted on S3 Bucket in AWS.

Students prepared serverless application based on AWS hosting platform with usage of following services:

  • IoT Core
  • Dynamo DB
  • Lambda
  • SNS (Simple Notification System)
  • API Gateway
  • S3 Bucket - Static Website Hosting

Application purpose is to automatically notify users of dangeroues levels of air pollution near the user location. User can register for an automatic newsletter by:

  1. Selecting a nearest user location at map below,
  2. Providing a Phone Number,
  3. Providing a Email Address,
  4. Clicking Register Button.

Once user click Register button, application will create subscription for newsletter and user has to confirm subscription through e-mail. Users are notified about dangerouse level of air pollution by e-mail.
